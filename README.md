## Description
This project consists of the design and development of a web portfolio as a final project for the course Web Layout with Style Sheets.
The main objective is to create a clear, responsive, and visually coherent structure for presenting creative work in a web environment.
The content shown is fictional and has been used for academic purposes and for front-end design and development practice.

## Objectives
- Design a portfolio with a clear structure and intuitive navigation.
- Apply advanced CSS styles for layout, responsive design, and visual details.
- Implement basic interactions with JavaScript to improve the user experience.
- Organize code in a readable and maintainable way.

## Features
- Navbar with section navigation
- Hero with main title and interactive button
- Smooth scroll to the projects section
- Sections for Works, About me, Skills
- Visual contact section
- Responsive design adapted to different screen sizes

## Technologies used
- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Pages

## Use of artificial intelligence
Artificial intelligence was used as a support tool during the development process, mainly to resolve technical issues, improve code structure, and explore possible interaction and animation solutions.
The final decisions regarding design, content organization, and visual adjustments were made manually.

## How to use this template
This project can be used as a starting point for creating a personal portfolio, adapting its structure and content.
1. Clone or download the repository.
2. Modify the texts and sections in the index.html file.
3. Adjust the visual styles in css/styles.css according to the project's needs.
4. Replace the dummy content with your own projects, images, and texts.
5. Publish the website using GitHub Pages or another similar hosting service.
